# Privacy Policy -- Discord Bot

## 1. Data Collection

This bot only collects the minimum data required for its functionality,
such as: - Usernames\
- User IDs\
- Guild IDs\
- Message IDs (only if required for commands or logging)\
- Invite information (if the bot supports invite tracking)

The bot does **not** collect personal information such as emails,
passwords, payment details, or any sensitive data.

## 2. Data Usage

Collected data is used only for: - Executing commands\
- Logging features\
- Server management functions\
- Improving bot performance

## 3. Data Storage

Data may be stored temporarily in logs or databases if required for bot
features.\
No personal data is shared or sold to third parties.

## 4. Data Removal

Server owners may request deletion of stored data at any time by
contacting the bot owner.

## 5. Third-Party Services

The bot does not use external analytics or tracking services unless
explicitly stated.

## 6. Compliance

The bot follows Discord's Developer Terms of Service and the Discord API
Policies.

## 7. Contact

For questions or data removal requests, please contact the bot owner via
Discord.
